# am-defect-dataset
# Additive Manufacturing Defect Dataset

## Overview
This dataset contains images and related data collected for the study of defect detection in additive manufacturing (AM) processes.

## Description
The dataset includes samples of defective and non-defective builds collected during experimental AM runs. The goal is to support research in automated defect detection and quality monitoring.

## Dataset Contents
- Layer-wise images
- Labeled defect categories
- Process parameter data (if applicable)

## Collection Method
Data was collected during controlled additive manufacturing experiments. Defects were manually annotated based on visual inspection and process observations.

## Intended Use
This dataset is intended for:
- Machine learning research
- Computer vision defect detection models
- Academic experimentation
