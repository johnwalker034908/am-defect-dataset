# Additive Manufacturing Defect Dataset

## Overview
This dataset contains images and related data collected for research on defect detection in Additive Manufacturing (AM) processes. The objective is to support the development of intelligent quality monitoring systems.

## Background & Motivation
Additive Manufacturing processes are prone to various defects such as porosity, layer delamination, and incomplete fusion. Manual inspection is time-consuming and inconsistent. 

This dataset was created to:
- Enable automated defect detection
- Support research in AI-driven quality control
- Improve reliability and monitoring of AM systems

The motivation behind this work is to contribute to the advancement of smart manufacturing using data-driven approaches.

## Location & Data Collection
The dataset was collected during controlled additive manufacturing experiments conducted in Australia. Data acquisition was performed under monitored laboratory conditions to ensure consistency and reliability.

Layer-wise images were captured during the build process. Defect regions were identified and annotated based on visual inspection and process observations.

## Dataset Contents
- Layer-wise build images
- Labeled defect categories
- (Optional) Process parameters such as temperature, speed, or power settings
- Annotation files for supervised learning

## Intended Use
This dataset is intended for:
- Machine Learning model development
- Deep Learning-based defect detection
- Computer Vision research
- Multimodal data fusion experiments
- Academic and industrial research in smart manufacturing

## Why This Dataset Was Created
Existing public datasets for additive manufacturing defect detection are limited in size or lack proper annotation. This dataset aims to provide structured, labeled data to accelerate research in automated inspection systems.
